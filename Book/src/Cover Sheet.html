This document defines a training curriculum for teaching programming to kids using the Python programming language, with an emphasis on graphics and game programming. It is not necessarily intended that every single item be completed. Rather, the curriculum helps define a set of skills to learn and points to resources that can be used to help develop those skills. Furthermore, while the curriculum has a rough order to it, this order isn't strict - it's expected that kids will jump around to different sections at different times.

New kids to the group should receive a physical printout of this curriculum. As kids complete items in the curriculum, the date column should be filled out. Bring your kid's copy of the curriculum to each meetup - doing so will help us plan appropriate exercises. If you see something on the curriculum that we haven't covered yet but would like to learn, just let us know at a meetup.
